How It Works

Get Started in Minutes

Simple, consistent API across all data sources.

01

Install

Install the parsers you need. Each parser is standalone — no huge dependencies.

02

Configure

Set your API tokens as environment variables. Most parsers work out of the box.

03

Parse

Call the parse() function. Returns clean list[dict] ready for your app.

04

Export

Output to JSON, CSV, or use the dataclasses directly.

Example: Notion Database → JSON 
# Install the Notion parser
pip install notion-client

# Set your token
export NOTION_TOKEN=secret_xxxx

# Parse a database
from parse_notion import query_database, to_json

pages = query_database("DATABASE_ID")
output = to_json(pages)

print(output)
Example: Excel → CSV 
# Install Excel parser
pip install openpyxl

# Parse Excel file
from parse_excel import parse, to_csv

records = parse("data.xlsx", sheet="Revenue")
output = to_csv(records)

print(output)
